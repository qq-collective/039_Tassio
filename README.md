# 🏺 #039 TASSIO v3.1

> *Transcribe & Archive System for Strategic Information Organization*
> 「理解のためのひと手間を、AIが肩代わりする。」

---

## 概要

ポッドキャストのエピソードを「聴く前に理解する」ための支援ツールです。

番組を選び、エピソードを選ぶだけ。Claude APIがエピソードの内容を要約・概念図化・専門用語解説し、「なるほど、こういう話か」という解像度を事前に作ります。理解できたら、あとは実際に聴くだけ。

ポッドキャスト視聴の代替ではなく、**視聴のきっかけと補助**として設計されています。

---

## 使い方

### 事前準備
- **ブラウザ**: Chrome 推奨
- **APIキー**: Anthropic の Claude APIキー（`sk-ant-...`）

### 手順

**STEP 1 — 番組を選ぶ**  
番組カードを確認して「エピソード一覧を取得」を押します。iTunes Search APIを使って最新50エピソードを自動取得します。

**STEP 2 — エピソードを選ぶ**  
一覧からエピソードをクリックして選択し、「TASSIOで解析」を押します。

**STEP 3 — 理解支援コンテンツを読む**  
以下が自動生成されます：

| 出力 | 内容 |
|------|------|
| TABLE OF CONTENTS | タイムスタンプ付き目次 |
| SUMMARY | 400〜600文字の要約 |
| CONCEPT MAP | Mermaidによる概念図 |
| GLOSSARY | 専門用語の解説（自動抽出） |
| NEXT STEP | Apple Podcastへの「聴く」ボタン |

**深掘りレポート（オプション）**  
目次の各トピックをクリックすると、そのトピックだけを掘り下げた解説レポートを生成できます。解説レベルは3段階から選択可能です。

- 小学生でもわかる
- 新聞記事レベル（デフォルト）
- 専門家向け

**PDF出力**  
「⬇ PDF」ボタンからブラウザのPDF印刷で保存できます。

- **エピソード取得**: iTunes Search API（無料・認証不要・CORS対応）

## 注意事項

- iTunes APIは最新50エピソードを取得します。それ以前のエピソードは手動入力モード（説明文ペースト）で対応してください
- Claude APIキーはページを閉じると再入力が必要です（localStorage非保存）
- iTunes APIの説明文に目次テキストが含まれる番組であれば、精度が高くなります

## 名前の由来

**TASSIO** は筒井康隆『旅のラゴス』に登場する記憶者の少年へのオマージュです。耳から聞いた物語を正確に再現する能力を持つ彼のように、TASSIOは音声コンテンツから情報の骨格を取り出し、理解可能な形に再構築します。

バクロニムとして：
> **T**ranscribe & **A**rchive **S**ystem for **S**trategic **I**nformation **O**rganization
