# 🎧 #039 mimikara.fm

> 音声を「耳から」入れて、頭で整理する。ポッドキャスト理解支援ツール。

---

## 概要

ポッドキャストや音声コンテンツを聴きながら、リアルタイムで文字起こし・要約・概念図・専門用語解説を自動生成するシングルHTMLアプリです。

「耳では聴けるけど、頭が追いつかない」というモヤモヤを解消します。

---

## 機能

| 機能 | 説明 |
|------|------|
| 🎤 リアルタイム文字起こし | 音声ファイルを再生しながらブラウザが自動でテキスト化 |
| 📝 要約・まとめ | 300〜500文字で内容をコンパクトに整理 |
| 🗺️ 概念図（チャート） | Mermaid記法で概念の関係をビジュアル化 |
| 📖 専門用語解説 | 難しい用語を最大8語、わかりやすく解説 |

---

## 使い方

### 事前準備
- **ブラウザ**: Chrome を使用してください（Web Speech API依存）
- **APIキー**: Anthropic の Claude APIキー（`sk-ant-...`）が必要です

### 手順

1. **APIキーを入力**  
   画面上部の入力欄に Claude APIキーを貼り付けます

2. **音声ファイルをアップロード**  
   ファイルをドラッグ＆ドロップ、または枠をクリックして選択  
   対応形式：`mp3` / `m4a` / `wav` / `ogg` / `aac`

3. **「文字起こし開始」ボタンを押す**  
   音声が自動再生され、リアルタイムで文字起こしが始まります  
   途中で止めたいときは「停止」ボタン

4. **「AIで分析」ボタンを押す**  
   文字起こしが完了したら Claude APIが分析を実行  
   要約・概念図・用語解説が自動生成されます

---

## 技術スタック

- **文字起こし**: Web Speech API（ブラウザ内蔵・無料・APIキー不要）
- **AI分析**: Anthropic Claude API（`claude-sonnet-4-20250514`）
- **チャート描画**: Mermaid.js v10
- **構成**: Vanilla JS / シングルHTMLファイル

---

## 注意事項

- Web Speech API は **Chrome 専用**です。Safari・Firefox では文字起こし機能が動作しません
- Claude APIキーは `localStorage` には保存されません。ページを閉じると再入力が必要です
- 長時間の音声（45分以上）はブラウザのメモリ制約により文字起こしが途切れる場合があります
- 音声認識の精度は環境・話者・専門用語の多さによって変わります
